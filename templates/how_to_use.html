Expert System (El Ex) for Theorem Proving
  • How to Use
  • Theoretical Background

Overview

This application can be used for automated theorem proving or disproving using either proff by contradiction or disproof by counterexample. Note that it currently only supports propositional logic with the ability to add first order logic support in the future.

How to use

To use this application, you need to write a list of axioms and a theorem to prove. The following operations can be used as operators:
  • ~: to perform NOT operation. Ex: ~x, means not x
  • +: to perform OR operation: Ex: x+y, means x OR y
  • *: to perform AND operation: Ex: x+y, means x AND y
  • >: to perform IMPLICATION: Ex: x+y, means x Implies y
  • brackets "(" or ")": Can also be used
  • Variables can have any characters but can't contain any of the above operation characters

Example Use (Proof by Contradiction)

  • x*y*z+var1
  • y*z
  • var1>>(x+y)
  • ~(~(z+x+var1))>>(x+y+z+var1)

Theorem to prove:

  • (~(x*z)>>var1) * (var1+z)
